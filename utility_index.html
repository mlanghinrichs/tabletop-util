Utilities

If there's a tool you would like that isn't listed here, contact me and I'd be happy to code almost anything for the site!


Encounter Roll Generator

This tool generates HP, initiative, and attack rolls for an encounter given basic monster stat inputs, then allows you to add players and sort by initiative to make an encounter reference in turn order.



Region Statistics

A generator for randomized statistics for a new region, on any scale from Settlement to Sector. Choose a population or size (or, soon, a preset) to produce professionals, population density, regional geography, and more.



Name Generator

This tool generates a set of random names given parameters of the names' structure. It also contains presets for popular fantasy, historical, and foreign naming conventions.
